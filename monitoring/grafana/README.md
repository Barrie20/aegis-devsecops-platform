# Grafana Dashboards

This directory contains Grafana dashboard configurations as JSON.

## Dashboards (Coming Soon)

- **API Health** — Request rate, latency percentiles, error rate
- **Infrastructure** — CPU, memory, disk, network
- **Security** — Failed auth attempts, suspicious activity
- **SLO Dashboard** — Service Level Objective tracking

## How to Import

1. Open Grafana UI (http://localhost:3000)
2. Go to Dashboards → Import
3. Upload the JSON file or paste the JSON content
